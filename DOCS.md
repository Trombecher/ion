# Documentation

## Primitive Types

- `num`
- `char`
- `str`
- `nil`
- `bool`
- `object`

## Composite Types

- `String`

## Markup

Markup elements start `<`. The token before `<` must be one of the following:

- `=`
- `==`
- `=>`
- `<`
- `<=`
- `<<`
- `<<=`
- `>`
- `>=`
- `>>`
- `>>=`
- `+`
- `+=`
- `-`
- `-=`
- `*`
- `*=`
- `**`
- `**=`
- `/`
- `/=`
- Line comment `//`
- Doc comment `///`
- `%`
- `%=`
- `|`
- `|=`
- `||`
- `||=`
- `&`
- `&=`
- `&&`
- `&&=`
- `^`
- `^=`
- `[`
- `{`
- `,`
- `;`
- `:`
- `!=`
